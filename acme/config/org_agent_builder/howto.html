Overview

All societies are now generated from CSV files. The society generation process consists of two parts. The generation of a facet configuration and the generation of a society (sometimes called plugins) configuration.
For all of the 1AD societies, facet files are easily generated from the CSV files using a simple script. For the UA societies however, a special tool: SocietyDesigner is required.

Facet Files

A facets file is a pre-configuration file that contains the basic information about each agent within that configuration. The information contained in each agent is used by a set of Ruby rules to determine which plugins should be inserted into the agent.
Facets are broken up into two seperate sections, the 1AD facets and the UA facets, however both facet files look similar.

1AD Facets

A script to generate a single 1AD facet file does not exist. Currently the base set of facet files are located in the 1ad directory in configs/org_agent_builder. The only reason these configurations would need to be re-generated would be if the CSV files changed. If that happens, running the script: make-facet-societies.sh will re-generate all of the facet configuration files.

UA Facets

To generate the UA facets, you must use the SocietyDesigner tool. This tool, located in this module allows the rolling up and un-rolling of the UA agents. For more information on using this tool, please consult the tool documentation.

Society Files

To run a configuration, you need a society file. Society files can be found in this module, config/societies/[ul|ua]. However if you need to generate a new society file, there exists a single script to assist. The script gen-plugins-xml.sh can be used to generate a single society.
The usage of this script is: 

% gen-plugins-xml.sh [society]
Where society is the partial path and name of the society that you wish to generate.
For example: 
% gen-plugins-xml.sh 1ad/tiny-1ad
Would generate a tiny-1ad society in the current directory using the facets file located in the 1ad/ directory.
The basic 1AD Societies are:
  • tiny-1ad
  • tiny-1ad-trans
  • small-1ad
  • small-1ad-trans
  • full-1ad
  • full-1ad-trans
If you wish to generate all societies in a zipped file, use the scripts:
create-ul-dist.sh or create-ua-dist.sh
