Welcome to 's Experiment Scheduler

The ACME Experiment Scheduler is an ACME Service plugin that allows you to queue ACME Experiment Defintion files for execution on the operator host of an UltraLog society. This control screen allows you to enable and disable this service, see which experiment is currently executing, and a list of queued experiments that are scheduled to run.
